import(_names, _basePath, _nginxStyle)

导入UXC组件
Parameters:
Name Type Description
_names string 模块名 或者模块下面的某个js文件(test/test1.js, 路径前面不带"/"将视为test模块下的test1.js) 或者一个绝对路径的js文件, 路径前面带 "/"
_basePath string 指定要导入资源所在的主目录, 这个主要应用于 nginx 路径输出
_nginxStyle bool 指定是否需要使用 nginx 路径输出脚本资源
